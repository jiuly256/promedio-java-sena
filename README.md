# PROMEDIO-JAVA

_Ejemplo sencillo de calculo de promedio con JAVA para SENA , calcula el promedio de 4 numeros_


### Instalacion


_hace lo siguiente para compilar_

```
java AppletPromedio.java

```

_Para visualizar_

```
	appletviewer AppletPromedio.Html
		
```
